netcopy is a very simple pastebin service written in Ruby.

## Getting started

    git clone https://github.com/mxhold/netcopy.git
    cd netcopy
    bundle

To run the tests, run:

    bin/rake

To start the server at <http://localhost:8080>, run:

    bin/unicorn
